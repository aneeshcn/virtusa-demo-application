# PlaygroundAngularApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 18.2.21.

## Development server

1. Try "npm install".
2. Once completed try to Run `npm install primeng primeicons` .
3. Thrn try "ng serve" for a dev server. Navigate to `http://localhost:4200/`. 

## Modes (left side bar):
I have used three modes on the left side:
 1. Customers
 2. Products
 3. Orders
